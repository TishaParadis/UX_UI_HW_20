My UX Skills

User Research

Using qualitative and quantitative research is a major step to designing for the user. User Research allows for greater insight to the purpose of any necessary design elements that will provide greater usability for all target users.

User Interface Design

User Interface Design combines the concepts of Interaction Design,Visual Design, and Information Architecture. This process is a crucial step that defines user accessibility.

Wireframing

Wireframing is critical to the design process. This step allows for the layout of a product and provides a visual of how a user might interact with the interface.
